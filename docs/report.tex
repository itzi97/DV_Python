\section{Introduction}

This report presents a comprehensive data visualization analysis of personal fitness activity data recorded over 5 years (2021–2026) using a Garmin Forerunner 245. The project demonstrates best practices from Unit 1 while answering three analytical questions through 5 visualizations using 3 libraries and the primary CSV dataset.

\subsection{Methodology}

The visualization process followed these steps:
\begin{enumerate}
\item Data exploration and cleaning using Pandas/NumPy
\item Question formulation based on data patterns
\item Library selection: Matplotlib (core), Seaborn (heatmaps), ipywidgets (interactivity)
\item Iterative chart refinement following Tufte's principles of data-ink ratio and clarity
\item Export to publication-ready PNGs for LaTeX integration
\end{enumerate}

\section{Dataset}

The dataset (\texttt{data/activities.csv}) contains 180 workouts across 31 columns from Feb 21, 2021 to Mar 30, 2026:

\begin{itemize}
\item \textbf{Key columns}: Date, Activity Type, Time, Distance, Avg HR, Max HR, Calories, Elevation Gain
\item \textbf{Activity types}: Running, Walking, Cycling, Strength, Swimming, etc.
\item \textbf{Data types}: Dates, numerics, categorical, geospatial (Title contains locations)
\end{itemize}

Initial AI-assisted schema analysis identified `--` as null indicator and suggested data types, all manually verified.

\subsection{Processing}

Key preprocessing steps (notebook Section 2):
\begin{itemize}
\item Replaced `--` nulls with NaN
\item Converted Time/Distance to numeric (regex parsing)
\item Extracted locations from Title
\item Created WHO-equivalent minutes from Avg HR thresholds
\item Weekly aggregation for time series analysis
\end{itemize}

Libraries: Pandas, NumPy, re (regex)

\subsection{Research Questions}

Three questions were formulated based on data exploration:

\begin{enumerate}
\item \textbf{Q1}: What types of activities dominate the dataset and their frequencies?
\item \textbf{Q2}: Do weekly totals meet WHO guidelines (150/300 min equivalents)?
\item \textbf{Q3}: How do activity patterns vary by location (Madrid vs others)?
\end{enumerate}

\section{Library Choices \& Best Practices}

\textbf{Matplotlib}: Core plotting, manual layouts, custom colormaps. Used for precise control (heatmaps, timelines).

\textbf{Seaborn}: Statistical plots (barplot, regplot). Simplified complex visualizations while maintaining customization.

\textbf{ipywidgets}: Interactivity for year comparison. Enhanced exploratory analysis.

\textbf{Best practices applied} (Unit 1):
\begin{itemize}
\item Color encoding follows perceptual principles (sequential for magnitude)
\item Minimal ink: removed grids, tight spacing
\item Clear hierarchy: prominent thresholds, trend lines
\item Accessibility: high contrast, clear typography
\end{itemize}

\section{Conclusions}

The analysis reveals:
\begin{itemize}
\item Running dominates training (47\%) but consistency improved 2025–2026
\item WHO targets met sporadically; 2026 shows upward trend but needs sustained 300+ min weeks
\item Location patterns suggest trail runs yield higher intensity
\end{itemize}
