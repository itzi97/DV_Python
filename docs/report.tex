\section{Introduction}

\par
The aim of this project is to explore data visualization using Python, from
selecting the data all the way to telling a story through it's visualization.

\subsection{Methodology}

\par
The project itself followed the following process to achieve the previously
mentioned goal:
\begin{enumerate}
	\item Dataset selection
	\item Data analysis and processing
	\item Questions derived from the dataset
	\item Visualization types needed to answer each question and storytell
	\item Further data processing needed for each visualization
	\item Library selection for each visualization
	\item Package visualizations within a combined story narrative that tells a
	      story about the data
\end{enumerate}

\subsection{Sources}

\par
Numerous sources are used throughout the project such as:

\begin{itemize}
	\item AI
	\item Documentation
	\item Forums
	\item Literature
	\item Lecture notebooks
\end{itemize}

\par
Regardless of the source, these are indicated through bibliography/references
or in the case of AI, an explicit note indicating the model and prompt used.

\subsection{AI Usage}

\par
A point was made to use AI in this project in a manner in which it's answers
are scrutinized and reviewed, to make sure hallucinations are not taken into
account and answers are not just implemented without a second thought. This
way, an active usage of AI is incentivized with certain restraints in place.

\par
AI being used for this project was also hosted locally, using smaller models to
do so to encourage reproducibility (since prompts are also included within
the notebook and report).

\section{Dataset}

\par
Data for this project was required to not have been selected by a different
student. To minimize the risk of this happening, activity from a Garmin
Forerunner 245 was chosen, reducing the chances of another student having the
same dataset. The data located in \texttt{./data/activities.csv} consists of:

\begin{itemize}
	\item \textbf{Shape}: 31 columns \& 180 rows
	\item \textbf{Date span}: 21 February 2021 - 30 March 2026 (5 years)
	\item \textbf{Data types}: Dates, numerics, categorical, geospatial (Title
	      contains locations)
	\item \textbf{Key columns}: Date, Activity Type, Time, Distance, Avg HR
	\item \textbf{Activity types}: Running, Walking, Cycling, Strength, Swimming,
	      etc.
\end{itemize}

\par
Having 31 columns allows for many different questions to be asked about the
dataset, specially since some of these columns allow for other data fields to
be extracted (activity intensity from Avg HR). The limited number of rows (180)
whilst still being a decent amount, allows it's content to be analyzed in a
more hands on way, being able to be more critical about each specific row and
actively engage in processing in a more granular fashion. The dates spanning 5
years back allows for a story to be told through time, and analyze factors
such as consistency. Lastly, varying data types allows for distinct types of
visualizations that require specific untis to be used such as coordinates
(from location).

\subsection{Analysis}

\par
Before asking questions about the dataset, it was further analyzed using both
the \texttt{df.info()} function to check how the data was imported into the
pandas dataframe. A small AI analysis was invoked using
qwen3.5:9b\autocite{qwen} for the prompt which was the following:

\begin{center}
	Read the linked CSV file and analyze it, create a table with the following
	fields:
	\begin{itemize}
		\item N: Column number
		\item Column: Column name
		\item Description: Content description (what the data in the column usually)
		      looks like), excluding null indicators
		\item Null indicator: Indicator that seems to be used for when the data is
		      null
		\item Data type: Suggested data type to be used when reading CSV in Pandas.
	\end{itemize}
\end{center}

\par
The table generated was then used as a starting point although it's content was
scrutinized and interpreted correctly, assessing whether or not the AI had
hallucinated when generating it. Some important conclusions reached with the
help of this analysis (done in section 2.1 of \texttt{data\_viz\_notebook.ipynb})
where:

\begin{itemize}
	\item Some columns weren't really needed or used, such as \textit{Favorite}
	\item "\texttt{--}" was used as a null indicator on most columns
	\item Correct data types for each column
	\item Some columns belonged to specific activity types, such as \textit{Avg.
		      Swolf} for swimming
\end{itemize}

\subsection{Processing}

\par
Key processing steps (section 3 of \texttt{data\_viz\_notebook.ipynb}):

\begin{itemize}
	\item Replaced `--` nulls with NaN
	\item Converted Time/Distance to numeric (regex parsing)
	\item Extracted locations from Title
	\item Created WHO-equivalent activity minutes\autocite[32]{WHO2020} from
	      Avg HR thresholds
	\item Weekly aggregation for time series analysis
	\item Activity Type simplification (treadmill running, trail running,
	      running are all running)
\end{itemize}

Libraries: Pandas\autocite{pandas}, NumPy\autocite{numpy}, re\autocite{re}
(regex)

Some of the data processing was effectuated in order to satisfy the data
requirements of some of the visualizations, which will be explained in the
following section.

\subsection{Research Questions}

Three questions were formulated based on data exploration:

\begin{enumerate}
	\item What types of activities dominate the dataset and their frequencies?
	\item How do activity patterns vary by location?
	\item Do recorded activities showcase "enough" weekly exercise?
\end{enumerate}

\subsubsection{Question 1: Activity Composition}\label{q1}

\par
Question 1 is a good starting point because it allows for the superficial
exploratory analysis of the data. This facilitates the understanding of what
activity types end up shaping most of the data, and is a good starting point
that is taken into account when further exploring other questions.

\subsubsection{Question 2: Geographic Patterns}\label{q2}

\par
Question 2 proceeds naturally from question 1, as it further dives into
understanding the nature of the data, in this case, classifying it into
geographic location rather than categorical activity types. Furthermore, it
also enables the use of different visualization methods due to the unique
geographic data type, location.

\subsubsection{Question 3: Weekly Activity Assessment}\label{q3}

\par
Question 3 is purposefully subjective to some degree, to allow for a variety of
approaches to visualizing and analysing the data, alongside more liberty when
it comes to choosing what narrative to pursue in storytelling. This allows for
many different charts to be used, and for data to be used in creative ways.

\subsection{Storytelling}

\par
A Garmin user tracks 5+ years of fitness data. Q1\sref{q1} and Q2\sref{q2}
reveal activity variety (locations and types). Q3\sref{q3} dives into health

\subsection{Colors and Themes}\label{colors}

\par
Colors and themes were chosen purposefully in order to make charts as simple
and readable as possible. The following design choices were made on purpose to
enhance clarity, readability and intent:

\begin{itemize}
	\item Green theme (emerald-500 tailwind\autocite{tailwindcolors} palette) +
	      teal accents, fitness/health
	      connotation, high contrast.
	\item Minimal ink: No grid/borders, no clutter, focus on data.
	\item Distinct category colors for pre-attentive processing (green vs. gray).
	\item Custom colormaps (sequential for magnitude, diverging for trends).
\end{itemize}

\par
These decisions were made in line with theory from the first unit.

\subsection{Q1 - Recorded Activity Types}

\par
The following options were considered: vertical bar chart, horizontal bar
chart, pie chart. The \textbr{horizontal bar chart} option was chosen due to
being able to be ordered and have the highest count at the top, making it very
easy to read. The other option would have been a pie chart with the walking and
running segments exploded, but that wouldn't have been appropriate considering
that they took up such a huge percentage of the total activity mix.

\par
For the dataset for this chart, data had to be aggregated to get counts by
activity type, creating another data set in doing so.

\begin{figure}[h]
	\centering
	\includegraphics[width=\textwidth]{../figures/q1_activity_types.png}
	\caption{Q1 Chart - Recorded Activity Types}
	\label{fig:q1}
\end{figure}

\par
\textbf{Insights}: In Fig.~\ref{fig:q1} one can tell that two specific activities
dominate the activity mix landscape (being walking and running). This is made
even clearer with the green color making them pop out whilst the grey color make
the other activities fall to the background. Styling is simple, with the chart
not having many distracting elements.

\subsection{Q2 - Maps}

\par
Maps were chosen for the second question to display variability in locations
for recorded activities. Two maps were chosen, due to most activities being
recorded in Madrid, but some activities being recorded elsewhere in Spain. This
allows for two different types of map to be used.

\par
For both maps, activities were aggregated by location, such as autonomous
community or municipality, creating a special data set per chart to be used.
Location itself was extracted from title, so even though it wasn't explicitly
given, a work around was used to be able to obtain it.

\subsubsection{Q2a: Activities by Autonomous Community}

\par
Very simple \textbf{choropleth map} that showcases the stark difference in
activities recorded in Madrid vs. other autonomous communities. The choropleth
map was chosen due to communities being able to be colored to show case
different amounts of activities recorded, although a scatter map could have
worked too.

\begin{figure}[h]
	\centering
	\includegraphics[width=\textwidth]{../figures/q2_spain_choropleth.png}
	\caption{Q2a Chart - Activities by Autonomous Community}
	\label{fig:q2a}
\end{figure}

\textbf{Insights}: Fig.~\ref{fig:q2a} shows a much darker shade of green for
Madrid than for other autonomous communities due to Madrid being more than
90\% of activities recorded.

\subsubsection{Q2b: Activities by Municipality (Madrid)}

\par
Due to Fig~\ref{fig:q2a} showcasing a stark difference between activities

\par
A \textbf{scatter map} fits well for this analysis due to circles showing how

\subsection{Q3 - Trends}

\par
This section dives into answering Q3\sref{q3} with creative usage of charts in
order to point out exercise trends throughout time periods, years and weeks.
Earlier data was cleaned to create the following columns:

\begin{itemize}
	\item Week and Year: Extracted from date time.
	\item Intensity: Extracted from Avg BPM.
	\item WHO Minutes: Extracted from time and using intensity to calculate it.
\end{itemize}

\subsubsection{Q3a: Overview Heatmap}

\par
This particular chart divides itself into three different subplots. This
reveals whether consistency and volume translate to outcomes.

\paragraph{Period Timeline}
This first chart grabs different activity periods which are periods during
which activities are recorded with less than 60 days between each, allowing to
see how consistent the exercise recording process has been. A horizontal bar
chart was utilized in a creative way, displacing horizontal bars to emulate
a \textbf{timeline} chart.

\paragraph{Frequency Heatmap}

A \textbf{heatmap} was chosen for this chart with a green color gradient showcasing
weeks during which activities were recorded, aggregating data by week and year.

\paragraph{Benefits Heatmap}

This last chart was also a \textbf{heatmap}, but in this case, the WHO Minutes were
filtered depending on whether or not a specific health benefit was obtained
that week or not, allowing us to see whether activity frequency translated
to health benefits.

\begin{figure}[h]
	\centering
	\includegraphics[width=\textwidth]{../figures/q3_weekly_overview.png}
	\caption{Q3a Chart - Overview Heatmap}
	\label{fig:q3a}
\end{figure}

\textbf{Insights}: Fig.~\ref{fig:q3a} shows how most activities were recorded
in \textit{bursts} rather than consistently throughout the years. Specific
periods are highlighted, with a current one in 2026. Most active weeks
translate to health benefits as shown when comparing middle and bottom
visualizations.

\pagebreak

\subsubsection{Q3b: Year-Specific Trends}

\par
Having seen the broad picture of the trend throughout the years, a zoomed in
chart follows, providing a year-specific view of how activities trended. Due
to a year having 53 weeks, a \textbf{lollipop} chart was chosen as opposed to
a bar chart, due to the chart being too visually cluttered if the latter option
was chosen.

\par
An interactive widget was coded in to facilitate the same chart
being able to be used for different years depending on what year is selected in
the drop down menu. Multiple thresholds were introduced, to keep the idea of
different health benefits being met if different minute amounts of activity
were achieved per week, these are easily seen due to color changing as
thresholds are met.

\begin{figure}[h]
	\centering
	\includegraphics[width=\textwidth]{../figures/q3_lollipop_2021.png}
	\caption{Q3b Chart - 2021 Activity Lollipop Chart}
	\label{fig:q3b}
\end{figure}

\textbf{Insights}: Fig.~\ref{fig:q3b} Shows how in the year 2021 most weeks are
empty due to no activities being recorded, but weeks with recorded activities
tend to always have some kind of health benefit.

\pagebreak

\subsubsection{Q3c: 2026 Weekly Activity Trend}

\par
Zooming into the most recent period, a \textbf{scatter plot} with a
\textbf{regression line} was chosen to show how current activity levels are
evolving over time, and whether the recent trajectory suggests improvement.

\par
As in previous charts, threshold lines are included to indicate whether weekly
activity reaches different health-benefit levels. Point colors change depending
on whether those thresholds are met. A confidence interval is also shown around
the regression line, making it easier to judge how closely the observed weekly
values follow the overall trend.

\begin{figure}[htbp]
	\centering
	\includegraphics[width=\textwidth]{../figures/q3_trend.png}
	\caption{Q3c Chart - 2026 Weekly Activity Trend}
	\label{fig:q3c}
\end{figure}

\textbf{Insights}: It is still too early to draw strong conclusions, but
Fig.~\ref{fig:q3c} suggests a slight upward trend in weekly activity. More
recent points are also more likely to reach at least one health benefit
threshold than the earlier ones.

\section{Conclusions}

\par
Visualizations reveal the following:

\begin{itemize}
	\item Running and walking dominate activities [Fig.~\ref{fig:q1}].
	\item Most activities recorded in Madrid, around a specific area [Fig.~\ref{fig:q2a}].
	\item Activities recorded in bursts (periods) [Fig.~\ref{fig:q3a}].
	\item WHO target for 150 minutes met when activities were recorded, 300+
	      minutes one met sporadically [Fig.~\ref{fig:q3b}].
	\item Current recorded period shows upward trend [Fig.~\ref{fig:q3c}].
\end{itemize}
