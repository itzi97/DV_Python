\subsection{Dataset}

\par
Data for this project was required to not have been selected by a different 
student. To minimize the risk of this happening, activity data from a 
\href{https://www.garmin.com/en-US/p/628939/}{Garmin Forerunner 245} was chosen, 
reducing the chances of another student having the same dataset.

The data set, located in \lstinline{./data/activities.csv}, consists of:

\begin{itemize}
  \item 31 columns \& 180 rows.
  \item Data spanning from: 21 February 2021 - 30 March 2026 (5 years).
  \item Varying data types (integers, floats, dates, time spans, locations, strings).
\end{itemize}
